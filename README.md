Built a full-stack web application that analyzes uploaded images and provides a probability-based estimate of whether the content is AI-generated or authentic.

🔧 Tech Stack
Frontend: Next.js (React, TypeScript, Tailwind CSS)
Backend: Node.js / Express API
API Integration: Third-party AI detection service
⚙️ Features
Upload and preview images (video support planned)
Real-time AI likelihood score with verdict (AI / Inconclusive / Real)
Confidence meter with dynamic UI feedback
Error handling and status tracking during analysis
📌 Current Status
Fully functional locally with backend integration
Image detection supported; video detection in progress
Deployment and scalability improvements planned
🎯 Goal

To evolve into a mobile-friendly tool where users can quickly verify media authenticity (e.g., via social media sharing workflows).
